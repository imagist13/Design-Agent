# code_ui 产品（Design-to-Code / UI Code Generators）

本目录收集将设计稿/组件化 UI 转换为代码的工具与框架，包括 Figma 转代码、自动化组件生成、设计系统编译器等。

示例收录要点：
- 项目名称
- 支持的前端框架（React / Vue / Angular / 原生）
- 输出质量（可维护组件、样式隔离、响应式支持）
- 项目链接与演示

示例条目：
- ScreenCoder —（如存在）将设计转换为可运行前端页面的工具（见子目录）

收录建议：标注支持的输入格式（Figma/Sketch/JSON）、示例输出仓库与使用限制（商业/开源）。
